Propagate the Gaussian distribution through a nonlinear function.

This method transforms the current Gaussian distribution p(x) through a given function y = h(x) by propagating information through the affine transformation. It returns a new Gaussian distribution representing p(y).

Template Parameters
FuncThe type of the function object.
Parameters
hThe function object representing the nonlinear transformation. It should take two arguments: the input vector and a reference to the Jacobian matrix. The function should return the transformed vector and populate the Jacobian matrix.
Returns
A new Gaussian distribution representing p(y).
