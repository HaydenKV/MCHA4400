\doxysubsection{Detailed Description}
Main SLAM visualization class implementing assignment requirements. 

Provides dual-\/pane visualization\+:
\begin{DoxyItemize}
\item Left pane\+: \doxylink{structCamera}{Camera} image + detected features + confidence ellipses ~\newline

\item Right pane\+: 3D world view + camera + landmarks + confidence ellipsoids
\end{DoxyItemize}

Based on Lab 8 VTK visualization foundations
