\doxysubsection{Detailed Description}
\doxylink{classMeasurement}{Measurement} class for Ar\+Uco tag SLAM with unique IDs. 

Handles\+:
\begin{DoxyItemize}
\item ID-\/based data association (trivial since tags have unique IDs)
\item 4-\/corner measurement model per tag
\item Grace period before deletion (10 frames while visible)
\item Uncertainty inflation for visible-\/but-\/unassociated landmarks 
\end{DoxyItemize}
