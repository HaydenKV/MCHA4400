template<typename Scalar = double>

Base class for Gaussian distributions.

This class provides a common interface and utility functions for Gaussian distributions.

Computes the log of the integral of the Gaussian distribution between two points.

This function returns the following value:

\[\log \int_a^b \mathcal{N}(x; \mu, \sigma^2) \,\mathrm{d}x \]

where \(\mu\) is the mean and \(\sigma\) is the standard deviation of the Gaussian distribution.

Scalar b,
Scalar & dldmu ) const

Computes the log of the integral of the Gaussian distribution between two points and its derivative with respect to the mean.

This function returns the following value:

\[l = \log \int_a^b \mathcal{N}(x; \mu, \sigma^2) \,\mathrm{d}x \]

and its derivative with respect to the mean:

\[\frac{\partial l}{\partial \mu} = \frac{\mathcal{N}(b; \mu, \sigma^2) - \mathcal{N}(a; \mu, \sigma^2)}{\int_a^b \mathcal{N}(x; \mu, \sigma^2) \,\mathrm{d}x} \]

where \(\mu\) is the mean and \(\sigma\) is the standard deviation of the Gaussian distribution.

Parameters
aThe lower bound of the integral.
bThe upper bound of the integral.
dldmuReference to store the derivative with respect to the mean.
Returns
The log of the integral value.
