Detailed Description

template<typename Scalar = double>
class Gaussian< Scalar >

Represents a Gaussian distribution.

This class implements a Gaussian (normal) distribution with support for various operations such as marginalization, conditioning, and transformation. It inherits from GaussianBase and provides additional functionality specific to Gaussian distributions.

Template Parameters
ScalarThe scalar type used for calculations, default is double.
