\doxysubsection{Detailed Description}
Class for measurements with \doxylink{classGaussian}{Gaussian} likelihood. 

This class represents measurements that have a \doxylink{classGaussian}{Gaussian} likelihood function. It inherits from the \doxylink{classMeasurement}{Measurement} class and adds functionality specific to \doxylink{classGaussian}{Gaussian} likelihoods.
