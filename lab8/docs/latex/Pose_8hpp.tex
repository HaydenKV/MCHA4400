Defines the \doxylink{structPose}{Pose} class for representing and manipulating 3D poses in $\mathsf{SE}(3)$. 

This file contains the definition of the \doxylink{structPose}{Pose} class, which represents 3D poses as elements of the Special Euclidean group $\mathsf{SE}(3)$. It provides functionality for pose composition, point transformation, and conversion between different representations (Eigen and Open\+CV).
